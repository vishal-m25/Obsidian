
Spring is a Java framework that was created with the aim of facilitating the development of applications, using the concepts of [^1][Inversion of Control] and Dependency Injection. Inside it, there's Spring MVC and Core Technologies (it's the base of Spring, inside it's the dependency injection package).



## MVC
M - model
V - view
C - controller

## Spring boot layers
 - controller  == response for the request
 - service  == business logic
 - repository
 - data source

## Routes:
	- static
	- group
	- dynamic
	- query
	- request payload
	- wild card










[^1]: ==a software design principle where the control flow of a program is managed by an external framework rather than the program itself==
